Ths is a component of the BUILD SI project where we aim to:
  1. develop models of application binary interface compatibility
  2. perform "splices" across binaries and libraries using spack
  3. for each splice, make a prediction with some model (a tool) if the binary will still work
  4. test the actual spliced binary to validate the prediction.
For more details about this experiment, see the Splice Experiment repository and spliced that runs the experiments.
